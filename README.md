## INSTALAÇÃO

# TERMUX
```bash
> Baixe O Termux (Óbvio)
> Abra E Digite:
> apt update && apt upgrade
> pkg install git
> git clone https://github.com/Shelbyx/AstroBot/
> cd AstroBot
> bash install.sh
> node index.js
> Escaneie O Código E Pronto!
```
